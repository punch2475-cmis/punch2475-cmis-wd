    Lists and Links

    For this assignment, I have to research about how to add lists and links in html. I have learn how to make an unorder list and an order list in html. Also how to add links in html and how to add the links back to my main web page.

    Requirement

    • Create a new folder called "listsandlinks" in your repository folder
    • Inside listsandlinks create a file called "index.html" and a file called "main.css"
    • In index.html create an ordered list of step by step instructions to create a new webpage on the production server.
    • Also create an unordered list of 5 or more links to websites with useful webdesign related information. In the nav section, create an unordered list of links with a link back to your main index page.
    • In your main index.html file in public_html, add an unordered list of links to the nav section with a link to "listsandlinks".
    • Add an article with a heading and a paragraph description of this assignment. Also add an unordered list that contains this list of requirements.
    • Turn in your work.

    Tables

    For this assignment, I learn how to create tables in html by using thead, tbody, and tfoot. In the second table I learn how to use the colspan arttribute, colspan is expand the column, the code is colspan="how many cell you want to expand to." In the third table, I learn how to use rowspan arttribute, rowspan is expand to other row, the code is rowspan="how many cell you want to exapan." Finally I learn how to link table file back to lists and links by using the code ../file name.

    Requirement

    • Create a new folder in your repository called "tables"
    • In the "tables" folder, create a new file called from your template "index.html"
    • Be sure to use thead, tbody, and tfoot elements in your tables.
    • Add html code to recreate the webpage below.
      • second table will require the use of the colspan attribute.
      • third table will require the use of the rowspan attribute.
    • In your repository index.html file, add an article with a heading, paragraph description and this list of requirements. Also add a link to the tables/index.html file.
    • Add a link from lists and links to this assignment as well.
    • Validate your code.
